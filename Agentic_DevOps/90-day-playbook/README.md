# 90-Day Playbook

This playbook is a governance-first path to running your first agentic AI workflow in a DevOps context. It covers one use case (CI build failure analysis in advisory mode), one team, and 90 days of structured work — from building your eval suite and defining permissions through validation and controlled expansion to a second use case.

This is NOT a vendor evaluation guide and NOT a full autonomous deployment plan. It's a proof of concept path that prioritizes governance, measurement, and organizational learning over speed.
