# Stack Reference

This is a point-in-time reference (March 2026) of the tooling landscape relevant to agentic DevOps. The space is moving fast — check release dates and maturity levels before citing any of this in planning documents or business cases. The reference is organized into four layers, from foundation infrastructure you already run to the agent intelligence layer where most current investment is happening. For tracking how the core standards (MCP, AGENTS.md, A2A) evolve, the authoritative source is the Linux Foundation's Agentic AI Foundation (AAIF), formed December 2025 with founding contributions from Anthropic, OpenAI, and Block, and platinum members including AWS, Google, Microsoft, Bloomberg, and Cloudflare.

See [tool-landscape.md](tool-landscape.md) for the full reference.
