# MLOps Deployment Workshop

KodeKloud MLOps Deployment Workshop — runnable lab content.

**Stack**: Documentation / Markdown + Python (Lambda, scikit-learn) + Terraform (AWS) + GitHub Actions

This tree is self-contained under `NFCU-session-2/`, which serves as the
workshop repository root. Repo-wide tooling (`pyproject.toml`, `Makefile`,
`.pre-commit-config.yaml`, `scripts/validate-local.sh`) and the single
`openspec/` install govern all sessions. Runnable-code checks are scoped to
`MLOps_Deployment_Workshop/` and `shared/`.
